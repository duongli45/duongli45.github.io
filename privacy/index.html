Privacy Policy - Scroll Escaper

Last Updated: March 20, 2026

Developer: Li Nguyen

1. Overview

Scroll Escaper is a screen time management app that helps you build healthier phone habits by balancing screen time with physical activity. This Privacy Policy explains what data we collect, how it is stored, and how it is used.

2. Data Stored on Your Device

Core app data — including monitored app selections, usage settings, daily usage records, and streak history — is stored locally on your device using an on-device database and shared App Group storage. This data is not transmitted to external servers.

3. Data Stored in the Cloud

To enable Squad features and sync your progress, certain data is stored on our servers using Google Firebase (Firestore):

Account Data:

Anonymous user ID (no email/password required)

Display name and avatar emoji

Monitoring status and last active timestamp

Squad Data:

Squad name, invite code, and members

Squad streak status and history

Daily Reports:

Steps walked, minutes earned, minutes used

Block count and limit adherence

No personal contact information, location data, or browsing history is included.

4. Analytics

We use Firebase Analytics to understand usage and improve the app:

Screen views and navigation

Feature usage events

App version, OS version, session duration

Anonymous app instance ID

This data is not linked to your identity or used for advertising.

5. HealthKit Data

Scroll Escaper reads step count data from Apple HealthKit:

Processed locally

Included in daily reports for squad sharing

Never sold or used for advertising

Used only for app functionality

6. Push Notifications

We use Firebase Cloud Messaging to send notifications. A device token is linked to your anonymous user ID. No personal information is associated.

7. Purchases & Subscriptions

Subscriptions are managed by RevenueCat. Payment processing is handled by Apple. We do not store payment information.

8. What We Do NOT Collect

Personal information

Location data

Photos or media

Contacts

Financial data

Advertising identifiers

9. Data Retention & Deletion

You can delete your account anytime from Settings. Uninstalling removes local data. Analytics data follows Firebase retention policies.

10. Children's Privacy

The app does not knowingly collect personal information from children under 13.

11. Changes to This Policy

We may update this policy. Continued use of the app means you accept the changes.
